# Security Policy

Clean Link Kit runs locally in the browser. The core package does not send URLs to any server.

## Supported versions

| Version | Supported |
| --- | --- |
| 0.1.x | Yes |

## Reporting a vulnerability

Please open a private security advisory on GitHub or email the maintainers if contact details are listed in the repository.

Include:

- A clear description of the issue
- Reproduction steps
- Impact
- Suggested fix, if you have one

Do not publicly disclose vulnerabilities before maintainers have had a reasonable time to respond.
