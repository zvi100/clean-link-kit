# Roadmap

Clean Link Kit should stay small, useful, and easy to trust.

## 0.1.x

- Keep the core package stable.
- Add more high-confidence tracking rules.
- Add more tests for real-world URL patterns.
- Improve accessibility checks in the web app.
- Add screenshots to the README.

## Later

- Add Firefox extension support.
- Add optional custom rule import and export.
- Add a public demo deployment.
- Improve extension settings without adding account sync.

## Not planned for now

- Backend service.
- Login system.
- Database.
- Analytics that track users.
- Automatic link rewriting without clear user control.
